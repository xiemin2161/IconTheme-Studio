        使用方法

        颜色绑定

        从图标颜色池中选色，并分别绑定到两个通用变量：--its-icon-primary（主色）和 --its-icon-secondary（辅色）。

        使用说明
        1. 在“主色变量”一行选择主色，在“辅色变量”一行选择辅色。
        2. 两行颜色联动：一行选中的颜色，在另一行会自动禁用。
        3. 选择完成后点击“绑定”，会立即生效并持久化。
        4. 点击“重置绑定”会清空当前绑定。
        5. 在你的项目里覆盖这两个变量即可统一换肤：
        :root {
  --its-icon-primary: #ACEE2A;
  --its-icon-secondary: #FFFFFF;
}

        辅色变量
        主色变量 `--its-icon-primary`
        辅色变量 `--its-icon-secondary`

        主题切换

        主题色（绑定到 `--its-icon-primary`）
        辅助色（绑定到 `--its-icon-secondary`）

        点击颜色后会立即更新页面图标颜色，用于实时查看主题切换效果。
