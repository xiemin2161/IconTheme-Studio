# IconTheme Studio 版本日志

## 当前版本

- 版本：`1.0.1`
- 项目名：`IconTheme Studio`

## 版本迭代记录

### v1.0.1

本次为 `1.0.0` 之后的迭代更新，主要聚焦品牌统一、构建流程统一、离线包可用性和颜色绑定交互优化。

- 统一品牌命名为 `IconTheme Studio`，并清理旧命名残留。
- 构建入口统一为 `npm run build`，保留标签处理流程。
- 修复离线下载包资源路径问题，补齐 `logo` 与 `favicon` 资源打包。
- 颜色绑定交互升级为主色/辅色双行选择，并支持联动禁用同色。
- 文档更新为完整流程说明，覆盖命名、构建、绑定、导出、接入全链路。

### v1.0.0

首个稳定版本，具备完整图标预览与主题变量能力。

- 支持 `FontClass` / `Symbol` / `Unicode` 三种图标接入方式。
- 支持名称 + 中文标签搜索，提升图标定位效率。
- 支持颜色绑定到 `--its-icon-primary` 与 `--its-icon-secondary`。
- 支持主题切换实时预览（针对 Symbol 模式）。
- 支持下载离线包，并根据是否绑定自动保留或隐藏主题切换入口。
